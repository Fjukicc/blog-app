### Setup Instructions

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Fjukicc/blog-app.git
   cd blog-app
   ```

2. **Install Dependencies**:
   With npm:
   ```bash
   npm install
   ```
   Or with Yarn:
   ```bash
   yarn install
   ```

4. **Run the Development Server**:
   Start the development server:
   ```bash
   npm run dev
   ```
   The app should now be running at `http://localhost:3000`.
